# Дизайн: fullscreen-сторис по залам из карточек блока «Какие тренировки входят в 19 990 ₸?»

Дата: 2026-07-20 · Статус: согласовано (выбор пользователя через опрос)

## Решения пользователя

1. **Паттерн**: fullscreen-сторис — тап по карточке типа открывает полноэкранную
   карусель в стиле блока «Чем отличается»: слайд на каждый зал, фото на весь
   экран, текст поверх, точки-пагинация.
2. **Структура**: 4 карточки-типа остаются (Силовые, HIIT, Reshape, Mind & Body);
   залы — слайды внутри сторис: Силовые → Upper Body + Legs, HIIT → Bootcamp +
   Metcon, Reshape и Mind & Body — по одному слайду.
3. **Mind & Body**: структура заложена, текст-заглушка до получения контента.

## Курирование контента (из полного ТЗ по зонам)

Показываем на слайде: название зала, 1 строка сути, направления как чипы
(с короткой пометкой), мета-строка про количество планов, одна большая цифра
(30+/25+ вариаций, до 1000 ккал), 1–2 продающих буллета (Синхронизатор, тренер,
Lagree). Убрано: повторяющиеся абзацы «не нужно считать кг» (сказано 1 раз на
слайд), развёрнутые перечисления планов.

## Реализация

- Всё в `src/pages/home/DesignHome.tsx` (страница-одиночка, стиль сохраняем):
  `ZONES`/`TRAINING_TYPES` данные + компонент `ZoneStoriesOverlay`.
- Оверлей: `fixed inset-0 z-[100]`, чёрный фон, `useSnapCarousel` для слайдов,
  точки внизу, крестик и Escape для закрытия, `document.body.overflow=hidden`
  на время показа.
- Фото залов: `public/design/zones/<key>.jpg` (upper-body, legs, bootcamp,
  metcon, reshape, mind-body) — пользователь добавит позже; до этого graceful
  fallback на тёмный фон (паттерн onError как в остальных секциях).
- Карточки типа становятся кнопками с аффордансом «Подробнее».
