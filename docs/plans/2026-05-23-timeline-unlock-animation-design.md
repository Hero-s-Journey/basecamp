# Дизайн: анимация «3 тренировки открывают чекап»

**Дата:** 2026-05-23
**Блок:** «14 дней подготовки» / `PreparationTimeline.tsx`
**Статус:** утверждён

## Цель

Визуально показать правило: чтобы открыть фитнес чекап, нужно пройти
минимум 3 тренировки в порядке **Верх → Низ → Кардио**. Кардио-слот
может быть Bootcamp или Metcon — анимация это демонстрирует, чередуя их.

## Решение (вариант A — таймлайн с замком и прогрессом)

4 овала в ряд (десктоп) / колонкой (мобилка):

1. `01 Upper Body` — верх
2. `02 Legs / Glute` — низ
3. `03 Bootcamp` ⇄ `Metcon` — кардио (подпись чередуется)
4. `04 День фитнес чекапа` — цель (замок → разблокировка)

### Цикл анимации (loop, ~5 сек)

1. **Сброс:** овалы 1–3 — «не пройдено» (приглушённые, с номером).
   Овал 4 — с иконкой замка, тусклый, без terracotta.
2. Овал 1 завершается: номер → галочка ✓, овал светлеет,
   стрелка к овалу 2 загорается.
3. Овал 2 завершается — то же.
4. Овал 3 завершается — то же.
5. **Разблокировка:** все 3 готовы → замок открывается,
   овал 4 заливается terracotta с пульсом-свечением.
6. Пауза ~1.2 сек → сброс. При сбросе кардио-подпись
   переключается (Bootcamp ⇄ Metcon).

### Состояния овала

- **Не пройдено:** `bg-white/[0.04] border-white/10`, номер серый.
- **Пройдено:** ярче, рамка светлее, вместо номера — ✓ в кружке.
- **Чекап locked:** иконка замка, тусклый.
- **Чекап unlocked:** `bg-terracotta border-terracotta`, glow-пульс.

## Техническая реализация

- Стейт-машина в `PreparationTimeline`: `step` (0–4) продвигается
  по таймеру (`setTimeout`), зацикливается; `cardioVariant`
  (0 = Bootcamp, 1 = Metcon) переключается каждый полный цикл.
- Анимация запускается, когда секция входит в зону видимости
  (IntersectionObserver), чтобы не крутиться вхолостую.
- Компонент `Pill` расширяется: пропсы для состояний
  pending/done и locked/unlocked, слот под иконку ✓ / 🔒.
- Десктоп и мобильный таймлайн используют одну логику и стейт.
- Cursor-following glow на овалах сохраняется.

## Тайминги

- ~900 мс на шаг (1→2→3→4).
- Пауза в состоянии unlocked ~1200 мс.
- Полный цикл ~5 сек, затем сброс + смена кардио-подписи.
