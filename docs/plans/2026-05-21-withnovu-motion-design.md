# Дизайн: моушен-редизайн в стиле withnovu.com

**Дата:** 2026-05-21
**Референс:** withnovu.com
**Статус:** утверждён

## Цель

Добавить современные scroll/entrance-анимации лендингу Basecamp по образцу
withnovu.com: анимированный hero с текстом поверх картинки, плавный navbar,
переходы между блоками.

## Контекст (текущее состояние)

- **Lenis** (плавный скролл) уже подключён через `SiteLayout` → `useLenis`.
- **GSAP** есть в зависимостях, но нигде не используется.
- Анимации сейчас — только `useReveal` (IntersectionObserver fade-up).
- Hero появляется без анимации входа; картинка сейчас в рамке.
- Navbar уже переключается прозрачный → стекло при скролле.

## Объём работ

### 1. Hero — текст поверх картинки

- Вернуть full-bleed фоновую картинку с тёмным градиентным оверлеем.
- Navbar остаётся прозрачным поверх hero.
- Контент по центру: заголовок → подзаголовок → стеклянная кнопка → адрес.
- **Вход при загрузке:** GSAP-таймлайн, элементы каскадом всплывают
  (fade + slide-up, stagger ~0.12s, ease `power3.out`).
- **Параллакс:** при скролле фоновая картинка движется медленнее текста
  (ScrollTrigger scrub); контент слегка растворяется.
- Бегущая строка остаётся снизу.

### 2. Navbar

- Поведение прозрачный → стекло при скролле — без изменений.
- Добавить мягкий fade-in логотипа и ссылок при загрузке.

### 3. Переходы между блоками

- Подключить **GSAP ScrollTrigger**, связать с Lenis
  (`lenis.on('scroll', ScrollTrigger.update)`) — иначе scrub дёргается.
- Заголовки секций и картинки въезжают с привязкой к скроллу + лёгкий
  параллакс на картинках (`CheckupResult`, `CTA`, карточки `Transformations`).
- Карточки оставить на текущем `useReveal` — не ломать рабочее.

## Технические решения

- Новый хук `src/hooks/useGsap.ts` — регистрация ScrollTrigger + связка с Lenis.
- GSAP уже в зависимостях — установка не нужна.
- Анимации сдержанные, без тяжёлых pin-секций (можно добавить позже).
