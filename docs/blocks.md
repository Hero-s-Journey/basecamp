# Блоки сайта Hero's Journey 4YOU

Лендинг — одна страница, 8 секций сверху вниз.

---

**1. Hero** — первый экран
Заголовок «Оцифруй своё физическое состояние», цена 19 990 ₸, кнопка «Записаться».
Справа — карусель радар-графиков Hero Score реальных клиентов (Бауржан, Ева, Гайнеш + слот «Ты»). Снизу бегущая строка.

**2. MetricsHub — «6 показателей одна оценка»** (`#metrics`)
Объясняет, что такое фитнес чекап. Большой гексагон-радар с итоговым баллом + 6 карточек показателей (состав тела, выносливость, мощность, жимовая/тяговая сила, сила ног). Карточки кликабельны — открывают детали.

**3. WhyMetrics — «Зачем проходить фитнес чекап?»** (`#why-metrics`)
Аргументация ценности. Заголовок «Без цифр ты тренируешься вслепую» + 3 карточки по аудиториям: новичок / тренируется без результата / уже в спорте.

**4. TrainingsIncluded — «Какие тренировки входят в 19 990 ₸?»** (`#trainings`)
Что даёт абонемент. Карусель из 5 фото-карточек залов: Upper Body, Legs/Glute, Bootcamp, Metcon, Reshape.

**5. PreparationTimeline — «14 дней подготовки»** (`#preparation`)
Путь к чекапу. Анимированный таймлайн: 3 тренировки (верх → низ → кардио) открывают «Фитнес чекап». Фоновое видео.

**6. Transformations — «Истории участников»** (`#transformations`)
Социальное доказательство. Карусель из 10 фото реальных атлетов, каждая карточка ведёт на Instagram reel.

**7. CTA — финальный блок** (`#cta`)
Призыв к действию «Готов узнать свои данные?», теги оффера, кнопка → страница оплаты.

**8. StudioStrip / FAQ** (`#studio`)
Аккордеон частых вопросов — снимает возражения.
